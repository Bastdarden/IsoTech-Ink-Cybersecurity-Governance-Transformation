# Enterprise Cybersecurity Governance & Risk Transformation Initiative

## Project Overview

This project simulates a real-world cybersecurity governance and risk transformation engagement conducted for a fictional organization, IsoTech Ink.

IsoTech Ink is a 200-employee digital printing and logistics company that relies heavily on cloud systems, third-party vendors, remote employees, customer payment information, and digital asset repositories.

The organization experienced cybersecurity governance challenges including inconsistent access controls, phishing attacks, weak vendor oversight, monitoring gaps, and incomplete incident response procedures.

## Project Outcome

The initiative established a structured cybersecurity governance program that improved organizational maturity, enhanced risk visibility, strengthened employee awareness, improved vendor oversight, and increased audit readiness.

This project demonstrates capabilities relevant to GRC Analyst, Cybersecurity Project Manager, Governance Analyst, Third-Party Risk Analyst, Security Program Coordinator, and Risk & Compliance roles.
